        Øv på å kjøre analyser i microdata.no-stil. Inneholder ikke ekte tall; et hobbyprosjekt, ikke laget av microdata.no.
      
    
      
    
        OpenStat er en motor for å stille spørsmål som kan besvares med kode og statistikk — og for å se og endre koden som gir svaret. Skriv i Python, R, DuckDB eller microdata-stil; kjør i nettleseren.
      
    
      
    
        SafeStat er en motor for å stille spørsmål som kan besvares med kode og statistikk — og for å se og endre koden som gir svaret. I tillegg kan SafeStat dele og gi tilgang til sensitive data: bare analyser som bruker et begrenset sett kommandoer slipper gjennom, slik at resultatene viser aggregater — aldri opplysninger om enkeltpersoner. Søsterversjonen
        OpenStat
        er en åpen, forenklet utgave uten innlogging.

    CSV omkodingsverktøy
    
        
      
    
      
    
        
    Genererer et recode-uttrykk fra korrespondansetabell i opplastet csv-fil.
    
        
    Formatet er basert på SSBs egne klassifikasjoner og kodelister: ssb.no/klass under «korrespondanser».
    
        
    Tabellen må være en semikolon-separert csv-fil med følgende kolonner:
    fra-kode, fra-navn, til-kode og til-navn
    
        
    Tabellen må også inneholde en første linje som beskriver de etterfølgende kolonnene. Om denne inneholder de tre verdiene ‘sourceCode’, ‘targetCode’ og ‘targetName’, så vil kolonnene under disse bli definert som hhv. fra-kode, fra-navn og til-kode. Om disse ikke er definert vil standardoppsettet fra over brukes. Kolonnen ‘targetCode’ kan unnlates om du ikke vil sette en label på den omkodede verdien.
    
        
    Eksempel
    
        
    Utsnitt fra korrespondansetabell mellom økonomiske regioner og kommuner. Denne tabellen har kolonnene i standardoppsett.
    
        
    'sourceCode'	;	'sourceName'	;	'targetCode'	;	'targetName'
'03001'	;	'Oslo'	;	'0301'	;	'Oslo'
'11001'	;	'Dalane'	;	'1101'	;	'Eigersund'
'11001'	;	'Dalane'	;	'1111'	;	'Sokndal'
    
        
    Dette vil generere recode-uttrykket
    
        
    recode variable ('03001' = '0301' 'Oslo') ('11001' = '1101' 'Eigersund') ('11001' = '1111' 'Sokndal')
    
        
    Om du heller vil omkode fra kommuner til økonomiske regioner og invertere tabellen, kan da verdiene i den første linjen, sourceCode/sourceName og targetCode/targetName bytte plass, og resten av verdiene stå som før
    
        
    'targetCode'	;	'targetName'	;	'sourceCode'	;	'sourceName'
    
        
    vil generere
    
        
    recode variable ('0301' = '03001' 'Oslo') ('1101' '1111' = '11001' 'Dalane')
    
        
    
          
          
        
    
        
        
      
    
      
    
        
      
    
    
    
  
    
  

  
    
    
    
      
    ⚙ Innstillinger
    
      
    
        
    
          
          
        
    
        
    
          
          
          
    Maks antall rader per import-kommando (10 – 100 000).
    
        
    
        
    
          
          
          
    Antall desimaler i tabellutdata. Tall mindre enn 1 får automatisk ekstra desimaler for å vise signifikante siffer.
    
        
    
        
    
          
          
          
    Standardvisning i tabulate. Kan overstyres per kommando med opsjonene nolabels / novalues, eller per script med // m2py: label-format=both|label|code.
    
        
    
        
    
          
          
          
    Om hver setning gjentas med >>> rett over outputen den lager. Kan overstyres per script med #options.show_commands=False.
    
        
    
        
    
          
          
        
    
        
    
          
          
    
            
            1.0×
          
    
          
    Hastigheten på opplesing av kommentarer (0.5 – 2.0).
    
        
    
        
    
          
          
          
    Ventetid etter blokker uten kommentarer i Kjør skrittvis. 0 = ingen pause.
    
        
    
        
    
          
          
          
    Når PÅ: matcher microdata.no — pseudonym-validering, type-sjekk, min populasjon 1000, små celler skjules, winsorisering, persentiler avrundes. Når AV: alle regler er slått av og du ser rå tall.
    
        
    
        
    
          
          
    
            
            
            
            
          
    
          
    Tersklene microdata.no bruker. Mock-populasjonen er mindre enn ekte data, så standardene (1000/5/10/10) stopper ofte mer enn de ville på ekte register — senk dem her ved behov. Skaleres ikke automatisk. Gjelder bare når avsløringskontroll er PÅ.
    
        
    
        
    
          
          
          
    Dynamisk = generer ved kjøring (standard). Statisk = last fra static_data/*.parquet (krever at filene er tilgjengelige). Per script: // m2py: data-source=static.
