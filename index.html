    Lag en API-nøkkel på console.anthropic.com og lim den inn her. Nøkkelen lagres kun i denne nettleseren; forespørsler går via appens server, men nøkkelen lagres eller logges ikke der. Forbruk belastes din egen Anthropic-konto.     Velkommen til OpenStat!
    
      
    
        OpenStat er en motor for å stille spørsmål som kan besvares med kode og statistikk — og for å se og endre koden som gir svaret. Skriv i Python, R, DuckDB eller microdata-stil; kjør i nettleseren.
      
    
      
    
        SafeStat er en motor for å stille spørsmål som kan besvares med kode og statistikk — og for å se og endre koden som gir svaret. I tillegg kan SafeStat dele og gi tilgang til sensitive data: bare analyser som bruker et begrenset sett kommandoer slipper gjennom, slik at resultatene viser aggregater — aldri opplysninger om enkeltpersoner. Søsterversjonen
        OpenStat
        er en åpen, forenklet utgave uten innlogging.

    ⚙ Innstillinger
    
      
    
        
    
          
          
        
    
        
    
          
          
          
    Antall desimaler i tabellutdata. Tall mindre enn 1 får automatisk ekstra desimaler for å vise signifikante siffer.
    
        
    
        
    
          
          
          
    Om hver setning gjentas med >>> rett over outputen den lager. Kan overstyres per script med #options.show_commands=False.
    
        
    
        
    
          
          
        
    
        
    
          
          
    
            
            1.0×
          
    
          
    Hastigheten på opplesing av kommentarer (0.5 – 2.0).
    
        
    
        
    
          
          
          
    Ventetid etter blokker uten kommentarer i Kjør skrittvis. 0 = ingen pause.
    
        
    
        
        
    Microdata
    
        
    
          
          
          
    Maks antall rader per import-kommando (10 – 100 000).
    
        
    
        
    
          
          
          
    Standardvisning i tabulate. Kan overstyres per kommando med opsjonene nolabels / novalues, eller per script med // m2py: label-format=both|label|code.
    
        
    
        
    
          
          
          
    Når PÅ: matcher microdata.no — pseudonym-validering, type-sjekk, min populasjon 1000, små celler skjules, winsorisering, persentiler avrundes. Når AV: alle regler er slått av og du ser rå tall.
    
        
    
        
    
          
          
    
            
            
            
            
          
    
          
    Tersklene microdata.no bruker. Mock-populasjonen er mindre enn ekte data, så standardene (1000/5/10/10) stopper ofte mer enn de ville på ekte register — senk dem her ved behov. Skaleres ikke automatisk. Gjelder bare når avsløringskontroll er PÅ.
    
        
    
        
    
          
          
          
    Statisk = last fra static_data/*.parquet (standard). Dynamisk = generer ved kjøring. Per script: // m2py: data-source=dynamic.
