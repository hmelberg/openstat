// Example: Praktisk gjennomgang – temakurs om inntekt - skatt - trygd
// Source: https://microdata.no/manual/eksempel/Kursmateriale/Praktisk gjennomgang – temakurs om inntekt - skatt - trygd/
// ===================================================================
// Denne praktiske gjennomgangen viser hvordan man bruker ulike typer inntekts- og trygdedata
// til å kartlegge arbeidsmarkedsstatusen til individer. Kursmaterialet ble presentert i et
// temakurs i 2022 og dekker to hovedtilnærminger for å analysere sysselsettings- og
// ytelsesmottakermønstre.
//
// Første del undersøker inntektsdata fra to kilder: SSBs inntektsstatistikk (INNTEKT_
// -variabler) og A-ordningen for arbeidsforhold (ARBLONN_-variabler). Disse datasettene gjør
// det mulig å analysere lønnsmønstre, yrkesinntekter og pensjonsgivende inntekter på tvers
// av ulike tidsperioder og demografiske grupper.
//
// Andre del introduserer trygdedata og forklarer hvordan årlige inntektsdata knyttet til
// arbeid, ytelser og støtteordninger kan tjene som indikatorer på tilknytning til
// arbeidsmarkedet. I tillegg demonstreres bruk av FD-Trygd-data sammen med kursdata fra NUDB
// og jobbstatus fra sysselsettingsregistre for å vurdere økonomisk og sosial deltakelse.
// ===================================================================

textblock
SDC-effekt på gjennomsnittstall:

Gjennomsnittstall blir gjennomgående lavere pga. at ekstremverdier toppkodes.
Effekten varierer etter hvordan verdifordelingene ser ut:

- Høyrevridd fordeling => større effekt
- Normalfordelt eller venstrevridd fordeling => mindre effekt

- Lønn og yrkesinntekter (årsbeløp): -1,7 - -2,2%
- Lønn og yrkesinntekter (månedsbeløp): -0,5 - -1,8%
   -- unntak: kontantlønn: -1,5 - -4,1%
- Samlet inntekt og skatt (årsbeløp): -2,9 - -6,6%
- Trygd (årsbeløp): -0,1 - -1,2%
- Husholdningsinntekt etter skatt (årsbeløp): -2,1 - -2,2%

Månedsbeløp og trygd har mindre spredning => mindre avvik

Totalinntekter har større spredning => større avvik

Kjønnsforskjeller blir noe lavere pga. menn i større grad har ekstremverdier som toppkodes
endblock
