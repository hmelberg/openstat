// ===================================================================
// Example: Analysere hovedjobber
// Source: https://microdata.no/manual/eksempel/Tema%20-%20Jobb-%20og%20inntektsanalyser/Analysere%20hovedjobber/
// ===================================================================
// Variabler med prefiks ARBLONN_ARB_ og ARBLONN_LONN_ inneholder informasjon om alle
// arbeidsforhold registrert gjennom A-ordningen. Disse datasettene opererer på jobb-/
// arbeidsforhold-nivå, ikke personnivå, slik at individer kan ha flere jobber samtidig - med
// flere observasjoner enn antall unike personer.
//
// Når man lager statistikk eller analyser av jobber på individnivå, fokuserer man typisk på
// spesifikke typer arbeidsforhold per person - som hovedjobben, jobben med høyeste
// stillingsprosent, jobben med lengst avtalt arbeidstid, eller jobben med høyest månedslønn.
//
// Dette eksempelet demonstrerer hvordan man identifiserer og analyserer jobber definert som
// hovedarbeidsforhold.
// ===================================================================

require no.ssb.fdb:31 as db

// Lager jobbdatasett av aktive jobber per 16/7 2023, og finner hovedjobber per individ
create-dataset hovedjobber
import db/ARBLONN_ARB_YRKE_STYRK08 2023-07-16 as yrke
import db/ARBLONN_ARB_STILLINGSPST 2023-07-16 as stillingspst
import db/ARBLONN_ARB_HOVEDARBEID 2023-07-16 as hovedjobb
import db/ARBLONN_ARB_ANSETTELSESFORM 2023-07-16 as ansettelsesform
import db/ARBLONN_ARB_ARBEIDSTID 2023-07-16 as arbeidstid
import db/ARBEIDSFORHOLD_PERSON as personid

textblock
Antall aktive jobber per 16/7 2023 i jobbdatasettet, fordelt på hovedarbeidsforhold og biarbeidsforhold:
endblock
tabulate hovedjobb, missing
keep if hovedjobb == '1'

textblock
Stillingsprosent og avtalt arbeidstid for alle hovedjobber:
endblock
summarize stillingspst arbeidstid
histogram stillingspst, bin(5) percent
histogram arbeidstid, bin(5) percent

textblock
Stillingsprosent og avtalt arbeidstid for alle hovedjobber, fordelt på ansettelsesform:
endblock
tabulate ansettelsesform, missing
tabulate ansettelsesform, summarize(stillingspst, arbeidstid)

// Aggregerer jobbdataene opp til personnivå og kobler på persondata for å kunne lage personstatistikk
collapse(mean) stillingspst arbeidstid, by(personid)

create-dataset personer
import db/BEFOLKNING_KJOENN as kjønn
merge kjønn into hovedjobber

use hovedjobber
textblock
Stillingsprosent og avtalt arbeidstid for alle hovedjobber, fordelt på kjønn:
endblock
tabulate kjønn, missing
tabulate kjønn, summarize(stillingspst, arbeidstid)
