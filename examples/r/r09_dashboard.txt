# Dashboard — fordelingsutforsker (R)

Normalfordelingen N(mu, sigma). R-fasaden har INGEN `ui_kpi()`/
`ui_markdown()`-byggere (kun python/mpy/brython har `ui.html`-elementer å
bygge kort av — spec §3/§6) — «KPI-kortene» under er derfor to vanlige
tekstlinjer fra `cat()`/`sprintf()`, ikke et eget kort-vokabular. Kurven er
et base-plott, fanget som bilde slik R-motoren alltid gjør det.

Kontrollcellen bruker `on_change = "all"` (den innebygde
"kjør hele dokumentet på nytt"-sentinelen) OG `sync_to =` på hver kontroll.
**R-begrensning** (ærlig pedagogikk): R sin "Kjør alle" har IKKE per-celle
output-attribusjon slik python/brython/mpy har (all utdata fra én kjøring
samles i én delt sone) — et `on_change` som kun retter seg mot
ENKELTCELLER (f.eks. `c("stats", "plott")`, slik andre motorers eksempler
gjør) etterlater derfor en foreldet duplikat-blokk ved siden av den ferske
etter første interaksjon. `on_change = "all"` unngår dette ved å kjøre HELE
dokumentet på nytt (samme mekanisme som toppmenyens Kjør-knapp) i stedet
for kun mål-cellene — litt tyngre per interaksjon, men alltid
duplikatfritt, og slider-verdiene bevares (siste synkroniserte verdi, ikke
default) takket være `sync_to`.

#%% r id=kontroller
mu <- ui_slider(-5, 5, value = 0, step = 1, label = "Forventning (mu)",
                name = "mu", on_change = "all", sync_to = "mu")
sigma <- ui_slider(1, 4, value = 1, step = 1, label = "Standardavvik (sigma)",
                   name = "sigma", on_change = "all", sync_to = "sigma")
terskel <- ui_play(-5, 10, value = -5, step = 1, label = "Terskel (spill av)",
                   name = "terskel", on_change = "all", sync_to = "terskel")

#%% r id=stats
# Ingen ui_kpi()/ui_markdown() i R (spec §3/§6) — to tekstlinjer i stedet
# for kort.
cat(sprintf("P(X > terskel): %.1f %%\n",
            100 * pnorm(terskel, mean = mu, sd = sigma, lower.tail = FALSE)))
cat(sprintf("97,5%%-kvantilen: %.2f\n", qnorm(0.975, mean = mu, sd = sigma)))

#%% r id=plott
# «Kjør alle» evaluerer setning for setning — plot() + abline() må derfor
# stå samlet i én { }-blokk, ellers feiler abline() med «plot.new has not
# been called yet» (samme fallgruve som rex_param_forms.txt sitt tips).
xs <- seq(mu - 4 * sigma, mu + 4 * sigma, length.out = 121)
{
  plot(xs, dnorm(xs, mean = mu, sd = sigma), type = "l",
       xlab = "", ylab = "", main = "Tetthet")
  abline(v = terskel, col = "red")
}
