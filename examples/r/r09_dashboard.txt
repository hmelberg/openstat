# Dashboard — distribution explorer (R)

The normal distribution N(mu, sigma). The R facade has NO `ui_kpi()`/
`ui_markdown()` builders (only python/mpy/brython have `ui.html` elements to
build cards from — spec §3/§6) — the "KPI cards" below are therefore two
plain text lines from `cat()`/`sprintf()`, not a separate card vocabulary.
The curve is a base plot, captured as an image the way the R engine always
does it.

R has per-cell output attribution in "Run all" (like python/brython/
micropython): each cell's text and plot land in the cell's own output slot,
and a targeted on_change (here: c("stats", "plott")) reruns ONLY the
target cells and replaces their slots — no stale duplicates.
on_change = "all" is still valid when the whole document should be rerun.

#%% r id=kontroller
mu <- ui_slider(-5, 5, value = 0, step = 1, label = "Mean (mu)",
                name = "mu", on_change = c("stats", "plott"), sync_to = "mu")
sigma <- ui_slider(1, 4, value = 1, step = 1, label = "Standard deviation (sigma)",
                   name = "sigma", on_change = "all", sync_to = "sigma")
threshold <- ui_play(-5, 10, value = -5, step = 1, label = "Threshold (play)",
                   name = "threshold", on_change = "all", sync_to = "threshold")

#%% r id=stats
# No ui_kpi()/ui_markdown() in R (spec §3/§6) — two text lines instead
# of cards.
cat(sprintf("P(X > threshold): %.1f %%\n",
            100 * pnorm(threshold, mean = mu, sd = sigma, lower.tail = FALSE)))
cat(sprintf("97.5%% quantile: %.2f\n", qnorm(0.975, mean = mu, sd = sigma)))

#%% r id=plott
# "Run all" evaluates statement by statement — plot() + abline() must
# therefore be kept together in one { } block, otherwise abline() fails with
# "plot.new has not been called yet" (the same pitfall as rex_param_forms.txt's tip).
xs <- seq(mu - 4 * sigma, mu + 4 * sigma, length.out = 121)
{
  plot(xs, dnorm(xs, mean = mu, sd = sigma), type = "l",
       xlab = "", ylab = "", main = "Density")
  abline(v = threshold, col = "red")
}
