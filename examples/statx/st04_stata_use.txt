# label: Stata — use (multiple datasets)
* Example: Stata – use (switch between datasets)
* The first block creates two datasets: 'people' and 'people2'.
* The second block demonstrates 'use NAME' to switch the active dataset.

#micro
require no.ssb.fdb:54 as fd
create-dataset people
import fd/BEFOLKNING_KJOENN as gender
import fd/INNTEKT_WLONN 2022-01-01 as income
create-dataset people2
import fd/BEFOLKNING_KJOENN as gender
import fd/INNTEKT_WLONN 2022-01-01 as income

#stata
use people
summarize income
use people2
tabulate gender
