# Events på HTML-elementer (MicroPython)

`ui.on(selector, event, funksjon, target=…)` binder en micropython-funksjon
til en HTML-event; returverdien rendres i `target`-elementet (tekst →
`<pre>`, DataFrame → tabell, plotly-figur → graf). Utelates `target`,
appendes resultatet i cellens egen output-slot. `ui.run_cell(selector,
event, celle_id)` kjører i stedet en navngitt celle på nytt.

Dette er **push-modellen**: knappen dytter en event inn i MicroPython, i
motsetning til `ui.slider`/`ui.dropdown` som er pull (cellen leser verdien).

#%% html
<button id="beregnBtn">Vis rader</button>
<button id="plotBtn">Tegn plott på nytt</button>
<div id="resultat"></div>

#%% micropython
import ui

def vis(evt):
    return iris.head(5)

ui.on("#beregnBtn", "click", vis, target="resultat")
ui.run_cell("#plotBtn", "click", "plot")

#%% micropython id=plot
import plotly_express_mpy as pe
pe.scatter(iris, x="sepal_length", y="petal_length", color="species")
