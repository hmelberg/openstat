# Events on HTML elements (MicroPython)

`ui.on(selector, event, function, target=…)` binds a micropython
function to an HTML event; the return value is rendered in the `target`
element (text → `<pre>`, DataFrame → table, plotly figure → chart). If
`target` is omitted, the result is appended to the cell's own output
slot. `ui.run_cell(selector, event, cell_id)` instead reruns a named
cell.

This is the **push model**: the button pushes an event into MicroPython,
unlike `ui.slider`/`ui.dropdown`, which are pull (the cell reads the
value).

#%% html
<button id="showBtn">Show rows</button>
<button id="plotBtn">Redraw plot</button>
<div id="result"></div>

#%% micropython
import ui

def show_rows(evt):
    return iris.head(5)

ui.on("#showBtn", "click", show_rows, target="result");
ui.run_cell("#plotBtn", "click", "plot");

#%% micropython id=plot
import plotly_express_mpy as pe
pe.scatter(iris, x="sepal_length", y="petal_length", color="species")
