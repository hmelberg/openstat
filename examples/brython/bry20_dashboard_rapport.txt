# Iris — datarapport

Klassikeren fra 1936: 150 blomster, tre arter, fire mål. Kort-raden under er
bygget med ETT `ui.html.div(style={"display": "grid", ...})`-kall som holder
tre `ui.kpi(...)`-kort og to figurer (`ui.image(fig)`, som gjenkjenner en
plotly-figur og rendrer den nativt i stedet for et bilde). Dette er
`ui.html`-layout-idiomet — du bygger rutenettet selv i python, i stedet for
å bruke cellens `cols=`-attributt (se bry10/bry17/bry18/bry22 for det).

#%% brython id=kort
import ui
import pandas_brython as pd
import plotly_express_brython as pe

vc = iris["species"].value_counts()
kake = pe.pie(values=list(vc.values), names=list(vc.index), title="Fordeling av arter")
hist = pe.histogram(iris, x="petal_length", title="Petal-lengde")

ui.html.div(
    ui.kpi(len(iris), unit="stk", label="Blomster"),
    ui.kpi(len(iris["species"].unique()), label="Arter"),
    ui.kpi(round(iris["sepal_length"].mean(), 2), unit="cm", label="Snitt sepal-lengde"),
    ui.image(kake),
    ui.image(hist),
    style={"display": "grid", "grid_template_columns": "repeat(3, 1fr)", "gap": "16px"},
)
