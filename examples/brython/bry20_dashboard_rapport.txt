# Iris — data report

The classic from 1936: 150 flowers, three species, four measurements. The
card row below is built with ONE `ui.html.div(style={"display": "grid",
...})` call that holds three `ui.kpi(...)` cards and two figures
(`ui.image(fig)`, which recognizes a plotly figure and renders it natively
instead of as an image). This is the `ui.html` layout idiom — you build the
grid yourself in Python, instead of using the cell's `cols=` attribute (see
bry10/bry17/bry18/bry22 for that).

#%% brython id=kort
import ui
import pandas_brython as pd
import plotly_express_brython as pe

vc = iris["species"].value_counts()
pie_chart = pe.pie(values=list(vc.values), names=list(vc.index), title="Distribution of species")
hist = pe.histogram(iris, x="petal_length", title="Petal length")

ui.html.div(
    ui.kpi(len(iris), unit="pcs", label="Flowers"),
    ui.kpi(len(iris["species"].unique()), label="Species"),
    ui.kpi(round(iris["sepal_length"].mean(), 2), unit="cm", label="Mean sepal length"),
    ui.image(pie_chart),
    ui.image(hist),
    style={"display": "grid", "grid_template_columns": "repeat(3, 1fr)", "gap": "16px"},
)
