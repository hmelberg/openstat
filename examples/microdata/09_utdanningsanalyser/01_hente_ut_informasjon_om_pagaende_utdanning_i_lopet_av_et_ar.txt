// Example: Hente ut informasjon om pågående utdanning i løpet av et år
// Source: https://microdata.no/manual/eksempel/Tema%20-%20Utdanningsanalyser/Hente%20ut%20informasjon%20om%20p%C3%A5g%C3%A5ende%20utdanning%20i%20l%C3%B8pet%20av%20et%20%C3%A5r/
// ===================================================================
// Kursdata har kurs som enhetsnivå (ikke person) - hver person kan ha flere kurstyper samtidig.
// Kurs er gitt ved kombinasjonen person x kurstype. Fordi kursdata ikke har personer som
// enhetstype, kan de ikke importeres direkte inn i persondatasett på standard måte. I stedet
// brukes merge til å etablere koblingen mellom kurs-id og personidentifikator i kursdatasettet.
//
// Arbeidsflyten består av tre steg: først kobles kurs-id mot person-id i kursdatasettet,
// deretter aggregeres til personnivå med collapse, og til slutt kobles resultatene inn i
// persondatasettet. Eksempelet demonstrerer uthenting av årlig utdanningsforløp for personer
// bosatt i Norge, med fokus på høyere utdanning (masternivå eller høyere).
// ===================================================================

// Kobler til databank
require no.ssb.fdb:30 as db

// Oppretter persondatasett for bosatte per 2020-01-01
create-dataset bosatte
import db/BEFOLKNING_KJOENN as kjønn
import db/BEFOLKNING_STATUSKODE 2020-01-01 as regstatus
keep if regstatus == '1'

// Henter personer som tar høyere utdanning i løpet av 2020
create-dataset kursdata
import-event db/NUDB_KURS_NUS 2020-01-01 to 2020-12-31 as kurstype
destring kurstype
keep if kurstype >= 700000 & kurstype < 900000

// Kobler på lenke mellom kurs-id og fødselsnumre
create-dataset lenke_kurs_person
import db/NUDB_KURS_FNR as fnr
merge fnr into kursdata

// Lager statistikk (collapser) over antall hendelser med høy utdanning per individ, og kobler dette på persondatasettet
use kursdata
collapse (count) kurstype -> ant_kurs, by(fnr)
merge ant_kurs into bosatte

// Lager tabell over antall personer som tok høy utdanning i løpet av 2020
use bosatte
generate utdanning_høy = 0
replace utdanning_høy = 1 if ant_kurs >= 1
tabulate utdanning_høy kjønn
