// A standard wide-format dataset is created through three steps: establishing a database connection
// using `require`, creating an empty dataset with `create-dataset`, and importing at least one
// variable using `import`.
//
// It is recommended to connect to the newest database version to access the latest variables and
// updates. A key constraint is that only one variable can be imported at a time when creating a
// wide-format dataset ("Man kan bare importere en variabel om gangen når man lager et wide-format
// datasett"). The import function performs two essential tasks: retrieving data observations for a
// specific measurement point and linking the data to the current population through a unique
// built-in unit identifier.
//
// After dataset creation, modifications are possible, including renaming datasets or variables,
// removing variables, or eliminating observations. As an additional tip, the `outer_join` option
// allows overriding the default left-join principle to retrieve all observations for a given time
// period, even for individuals not previously in the dataset population.
// ===================================================================
