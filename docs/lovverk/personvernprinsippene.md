# Personvernprinsippene — oversikt

**Kilde:** Helsedirektoratet, Normen, Faktaark 57 — Personvernprinsippene
https://www.helsedirektoratet.no/normen/personvernprinsippene-faktaark-57

**Lovgrunnlag:** Personvernforordningen (GDPR) art. 5

## De syv prinsippene

1. **Lovlig, rettferdig og åpen behandling** — virksomheten skal opptre åpent og
   rettferdig overfor den registrerte ved behandling av personopplysninger.

2. **Formålsbegrensning** — personopplysninger kan bare brukes til de spesifikke,
   uttrykkelig angitte og berettigede formålene de er samlet inn for.
   Se [formalsbegrensning.md](formalsbegrensning.md).

3. **Dataminimering** — bare nødvendige personopplysninger skal samles inn og
   behandles. Se [dataminimering.md](dataminimering.md).

4. **Riktighet** — personopplysninger skal være korrekte og oppdaterte.

5. **Lagringsbegrensning** — personopplysninger skal ikke lagres lenger enn
   nødvendig. Se [lagringsbegrensning.md](lagringsbegrensning.md).

6. **Integritet og konfidensialitet** — data skal beskyttes mot uautorisert
   tilgang og misbruk.

7. **Ansvarlighet** — den dataansvarlige skal kunne dokumentere etterlevelse
   av de øvrige prinsippene.

## Anvendelsesområde

Prinsippene gjelder all behandling av personopplysninger i helse- og
omsorgssektoren, både helsehjelp, personal­administrasjon og forskning.
