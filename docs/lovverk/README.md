# Lovverk og offisielle retningslinjer

Samlemappe for lovtekst, faktaark og offisielle retningslinjer som er relevante
for personvern, dataminimering og bruk av mikrodata i forskning.

Innholdet brukes som referansegrunnlag for AI-baserte vurderinger av script
(dataminimering, disclosure-control) og som dokumentasjon for brukere som
ønsker å forstå hjemmelsgrunnlaget bak m2py sine sjekker.

## Tilføyninger senere

Aktuelle dokumenter å legge til:

- Personvernforordningen (GDPR) art. 5 — full ordlyd fra Lovdata

- Helseregisterloven § 6 — graden av personidentifikasjon

- Microdata.no sine egne brukerregler / retningslinjer for disclosure-control
- Datatilsynets veileder om dataminimering
- EDPB-retningslinjer om vitenskapelig forskning

## Merknad om sitater

Tekstutdragene i denne mappen er hentet fra offentlig publiserte kilder
(primært Helsedirektoratet) via automatisert nedlasting og sammendrag.
Direkte sitater er gjengitt i hermetegn, men skal verifiseres mot
originalkilden før de brukes i juridiske eller publiserbare sammenhenger.
Kildelenker står øverst i hver fil.
